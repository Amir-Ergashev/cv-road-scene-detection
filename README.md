# CV Project: детекция и классификация объектов дорожной обстановки

Учебный проект полного цикла в области компьютерного зрения.
Выполняется в рамках учебной практики БВТ.

## Предметная область

Интеллектуальные транспортные системы — детектирование и классификация
объектов на изображениях дорожной обстановки (автомобили, пешеходы,
велосипедисты, светофоры, дорожные знаки).

## Задачи проекта

- детектирование объектов на изображениях;
- классификация объектов;
- сравнение не менее 5 современных моделей.

## Используемые модели

1. YOLOv8
2. Faster R-CNN
3. SSD
4. EfficientDet
5. DETR

## Датасет

Подвыборка COCO (классы: car, person, bicycle, traffic light и др.).
См. `data/raw/README.md` после загрузки данных.


## Установка

```bash

cd cv-project-template
pip install -r requirements.txt
```

## Запуск

